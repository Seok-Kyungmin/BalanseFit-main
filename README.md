## 개발 목표 
SprnngBoot 기반 프로젝트와 MyBatis, Lombok Framework의 이해, MariaDB 사용, 반응형 웹페이지 구축

## 기술 스택
- Framework : Spring-Boot, MyBatis
- Database : MariaDB
- Front-end : JSP, HTML, CSS, JavaScript
- Cloud : AWS EC2, RDS
- Protocol : WebSocket
- API : 식품영양성분DB API, Kakao API, Naver Papago 번역 API, 언어감지 API
